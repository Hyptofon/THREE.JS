##  Використані технології

  * **Three.js**: Потужна JavaScript-бібліотека для створення та відображення 3D-графіки у веббраузері. Вона дозволяє легко працювати з геометрією, матеріалами, освітленням та анімаціями.
  * **Lil-GUI**: Легка та проста у використанні бібліотека для створення **налагоджувального інтерфейсу (Debug UI)**. Вона дозволяє створювати панелі керування з повзунками, кнопками, перемикачами та іншими елементами, що динамічно змінюють властивості об'єктів у сцені.
  * **GSAP (GreenSock Animation Platform)**: Бібліотека для створення високопродуктивних анімацій. У цьому проєкті вона використовується для плавного повернення об'єктів до їхніх початкових позицій, обертань та масштабів.
  * **OrbitControls**: Допоміжний клас з Three.js, який дозволяє керувати камерою за допомогою миші, обертаючи, масштабуючи та панорамуючи її.

## Основні можливості

  * **Різноманітні об'єкти**: На сцені розміщені чотири фігури — куб, сфера, тор та конус, кожна з яких має свій колір та матеріал.
  * **Інтерактивне керування**: За допомогою Lil-GUI можна динамічно змінювати наступні властивості для кожної фігури:
      * **Позиція** (`x`, `y`, `z`)
      * **Обертання** (`x`, `y`, `z`)
      * **Масштаб** (`x`, `y`, `z`)
      * **Колір**
      * **Видимість**
  * **Керування освітленням**: Можна регулювати інтенсивність навколишнього (`AmbientLight`) та спрямованого (`DirectionalLight`) світла.
  * **Керування сценою**: Дозволяє змінювати колір фону сцени.
  * **Анімації**:
      * **Автоматичне обертання**: Увімкнення автоматичного обертання для всіх фігур з можливістю налаштування швидкості.
      * **Ефект "стрибка" (`bounce`)**: Вмикає анімацію, де фігури плавно "стрибають" вгору та вниз, використовуючи функцію `Math.sin`.
  * **Скидання параметрів**: Кнопки для швидкого повернення позиції, обертання та масштабу всіх об'єктів до початкових значень.
