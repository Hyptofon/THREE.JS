# 🌌 Three.js Interactive 3D Sandbox

Цей проєкт — інтерактивний 3D-додаток, побудований з використанням **Three.js**. Він є платформою-пісочницею, що дозволяє користувачам досліджувати 3D-середовище від першої особи, взаємодіяти з геометричними фігурами та в реальному часі налаштовувати властивості освітлення, PBR-матеріалів і текстур. Проєкт має масштабовану модульну архітектуру та готовий до розвитку.

## 🚀 Основні можливості

- **FPS Керування (First-Person)**: Імплементовано управління камерою у стилі 3D-ігор. Клік мишкою захоплює курсор (на базі PointerLock API), а пересування відбувається за допомогою `W`, `A`, `S`, `D` з гладкою фізикою інерції та тертя.
- **Модульна Архітектура**: Проєкт побудований на професійній компонентній базі (окремі класи для `Camera`, `Renderer`, `World`, `HUD` та об'єктів), що забезпечує легке додавання нових фігур і логіки.
- **Динамічний Debug UI (Glassmorphism)**: Панель `lil-gui` оформлена у сучасному стилі "матового скла". Вона автоматично синхронізується зі станом об'єктів на сцені, підтримуючи зміну текстур, координат, PBR-параметрів (Roughness, Metalness) та кольорів.
- **PBR Матеріали та Текстури**: Підтримка `MeshStandardMaterial`, `MeshToonMaterial` та `MeshPhongMaterial`. Інтегрований `AssetLoader` динамічно вантажить текстури (Neon, Lava, Galaxy тощо).
- **Fullscreen Mode**: Підтримка повноекранного режиму для максимального занурення через подвійний клік.
- **Smooth GSAP Анімації**: Всі об'єкти можна повернути до початкового стану однією кнопкою "Reset" — скидання координат, масштабів, обертів і кольорів відбувається плавно завдяки бібліотеці GSAP.

## 🛠 Використані технології

* **Three.js**: Візуалізація 3D-графіки, освітлення, PBR та робота з камерами.
* **Lil-GUI**: Зручна панель управління параметрами сцени у реальному часі.
* **GSAP**: Плавні анімації та інтерполяція значень (easing).
* **Vite**: Надшвидкий збирач (bundler) локального середовища розробки.

## 📂 Структура проєкту

Проєкт повністю відповідає стандартам серйозних Production-додатків:

```text
├── public/                 # Статичні файли (текстури .png)
├── src/
│   ├── Utils/
│   │   ├── EventEmitter.js # Патерн Observer для безшовного зв'язку класів
│   │   ├── Time.js         # Ігровий цикл (requestAnimationFrame)
│   │   └── Sizes.js        # Відстеження ресайзу вікна
│   ├── UI/
│   │   ├── Debug.js        # Налаштування lil-gui
│   │   └── HUD.js          # Візуальний екран-підказка для гравця
│   ├── World/
│   │   ├── Figures/        # Усі 3D-фігури успадковуються від BaseFigure.js
│   │   │   ├── BaseFigure.js
│   │   │   ├── Cone.js
│   │   │   ├── Cube.js
│   │   │   ├── Sphere.js
│   │   │   ├── Torus.js
│   │   │   └── TorusKnot.js
│   │   ├── AssetLoader.js  # Словник та завантажувач усіх текстур сцени
│   │   ├── Camera.js       # Налаштування PerspectiveCamera
│   │   ├── Environment.js  # Світло та фон
│   │   └── World.js        # Ініціалізація та оновлення сцени
│   ├── Controls/
│   │   └── FPSControls.js  # Фізика руху WASD + PointerLock 
│   ├── Application.js      # Головний Оркестратор / Singleton
│   ├── Renderer.js         # Налаштування та запуск WebGLRenderer
│   └── config.js           # Централізоване управління кольорами та стилями
├── style.css               # Дизайн HTML-елементів (HUD, lil-gui)
├── script.js               # Точка входу застосунку (розмір 6 рядків)
└── index.html              # Базовий HTML-документ
```

## 🎮 Як грати та керувати

1. Натисніть **Лівою Кнопкою Миші** по екрану, щоб захопити курсор та увійти у режим польоту.
2. Рухайте мишкою для огляду навколо.
3. Використовуйте клавіші **W, A, S, D** для вільного польоту по сцені.
4. Натисніть **ESC**, щоб звільнити курсор і натискати на елементи панелі меню (справа).
5. Використовуйте **Double-Click** (подвійний клік) у будь-якому місці сцени, щоб увійти або вийти з повноекранного режиму.
6. Використовуйте **Debug UI**, щоб:
   - Відкрити блок будь-якої фігури (напр. *Cube (Standard)*).
   - Змінити її колір, текстуру (зі списку), координати або розміри.
   - Натиснути кнопку **Reset**, щоб фігура плавно відновила початковий вигляд.

## 🚀 Встановлення та Запуск

Переконайтеся, що у вас встановлений [Node.js](https://nodejs.org/).

1. Скопіюйте проєкт на свій ПК.
2. У терміналі відкрийте папку проєкту.
3. Встановіть залежності командою:
   ```bash
   npm install
   ```
4. Запустіть сервер для розробки:
   ```bash
   npm run dev
   ```
5. Відкрийте посилання (зазвичай `http://localhost:5173/`), яке видасть консоль, у вашому браузері.
